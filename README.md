# 🚸 Child Labor Tracker – Bangladesh

##Live APP: https://child-labor-tracker-bangladesh-y8b9e48e7ezwi5yx8vmhie.streamlit.app/

##📌 Project Overview

Child Labor Tracker is an interactive data dashboard that analyzes child labor patterns across different regions of Bangladesh.

The project explores how child labor varies by:

* 📍 Division
* 🏭 Sector (Agriculture, Industry, Services)
* 👦 Gender

It helps identify **high-risk areas and vulnerable groups**, supporting awareness and data-driven decisions.

---

## 🎯 Objectives

* Analyze child labor distribution across Bangladesh
* Identify high-risk regions and sectors
* Compare gender differences in child labor
* Build an interactive dashboard for exploration

---

## 📊 Features

* 📍 Division-wise analysis
* 🏭 Sector comparison (Agriculture, Industry, Services)
* 👦 Gender-based insights
* 📊 Interactive charts and filters
* ⚠️ Highlight high-risk regions

## 📊 Dataset

The dataset includes:

* Division
* Year
* Sector (Agriculture, Industry, Services)
* Gender
* Child Labor Rate (%)

---

## 💡 Key Insights

* Agriculture sector shows the highest child labor rates
* Boys are more involved in labor than girls in most regions
* Certain divisions consistently show higher risk
